# HHCraft
Modpack d'HikuTXW
Bienvenue sur le GitHub de mon modpack : HHCraft (HikuHypeCraft),
ce modpack que moi, HikuTXW j'ai créé, voulais que ce soit un modpack
bien rempli pour bien s'amuser avec ses amis, grâce à Essential Mod,
vos amis peuvent vous rejoindre sure un monde solo (maximum 4 personnes sur un monde)
et profitez d'un chat vocal et bien encore !
La version lite peut tourné sur, à peut prêt tout les ordinateur si ceci
n'est pas assez performant pour lancer la version normal.

Je vous souhaite un excellent jeu !
